# 赛博朋克·霓虹坦克大战（Cyberpunk Tank Battle）游戏说明书

欢迎来到**赛博朋克·霓虹坦克大战**！本游戏是一个基于 **FastAPI 后端** 和 **HTML5 Canvas 前端** 构建的、高度抛光的复古未来主义街机坦克射击游戏。

它包含了绚丽的霓虹光效、物理粒子爆炸、屏幕震动、复古 CRT 显像管滤镜，并搭载了**完全自主合成的 Web Audio API 音效引擎**（无需下载任何音频文件）以及功能完备的**地图编辑器**，支持本地**双人联机合作**。

---

## 🚀 快速启动指南

游戏的后端服务器已在后台为你启动，你可以直接在浏览器中输入以下地址开始体验：
👉 **[http://localhost:8000](http://localhost:8000)**

> [!TIP]
> ### 端口冲突解决（Address already in use 报错排查）
> 如果你手动在终端运行 `python fast.py` 时遇到了 `Address already in use (Errno 48)` 的错误，这是因为**后台已有一个正在运行的后台服务进程占用了 8000 端口**。
> - **解决方法 1**：无需在终端重复运行，直接打开浏览器访问 `http://localhost:8000` 即可开始游戏。
> - **解决方法 2**：如果你想在终端前台运行并查看日志，可以先杀掉后台进程（或使用 `lsof -i :8000` 查出 PID 并 `kill -9` 该 PID），然后重新运行 `python fast.py` 即可。

## 🎮 游戏操作按键

游戏设计了高效的多按键防卡死输入缓冲，支持两位玩家在同一块键盘上流畅地同时移动和开火。

### 🔵 玩家 1 (Player 1 - 霓虹青色战机)
- **向上移动**：`W`
- **向下移动**：`S`
- **向左移动**：`A`
- **向右移动**：`D`
- **发射激光**：`Space (空格键)` *(升级到 2 级后可按住连发)*

### 💗 玩家 2 (Player 2 - 霓虹粉色战机 - 双人模式)
- **向上移动**：`↑` (方向键上)
- **向下移动**：`↓` (方向键下)
- **向左移动**：`←` (方向键左)
- **向右移动**：`→` (方向键右)
- **发射激光**：`Enter (回车键)`

### ⚙️ 系统控制
- **暂停/继续游戏**：`Escape (ESC 键)`

## 📦 项目代码结构

我们采用纯原生现代 ES6 模块化方案，保持项目极度轻量、快速且易于扩展维护：

```
antigravity/code/
├── fast.py                 # FastAPI 后端，用于挂载静态目录并路由主页
└── static/                 # 静态前端资源目录
    ├── index.html          # HTML 主骨架，包含各类菜单和半透明 HUD 面板
    ├── style.css           # 赛博朋克视觉特效、毛玻璃卡片、CRT 扫描线、震动动画
    └── js/
        ├── config.js       # 游戏物理参数、霓虹色彩令牌以及 5 个精美关卡的数据结构
        ├── audio.js        # 声音管理器，通过 Web Audio API 动态合成所有音效及节奏背景乐
        ├── map.js          # 地图绘制器，实现 2x2 子砖块碰撞摧毁以及滑动冰面物理
        ├── entities.js     # 包含坦克基类、子弹、4类不同AI、7种道具及粒子火花
        ├── editor.js       # 画笔涂鸦、LocalStorage 地图读写以及地图合理性验证规则
        └── game.js         # 主渲染循环器、输入捕获、全屏震动及关卡敌人波次控制
```

## 💎 核心视觉特色与手感控制

1. **纯代码 Web Audio API 声音合成器**：
   无需下载任何庞大的 `.mp3` 或 `.wav` 音频文件。子弹的激光滋滋声（频率扫频）、坦克的引擎轰鸣声、核弹爆破低音炮、关卡清空或游戏失败的复古和弦旋律，全部在 JavaScript 里通过 Oscillator 和 NoiseBuffer 动态计算渲染。
2. **经典 2x2 微型砖墙碰撞系统**：
   高度还原经典红白机《坦克大战》的砖墙物理。每个 24x24 像素的网格被分为四个 12x12 像素的子砖块。你的子弹能削掉砖墙的任意一角，这让你能打出狭窄的弹道，或是躲在半截墙壁后打冷枪。
3. **沉浸式全屏震动特效 (Screen Shake)**：
   每次有坦克被摧毁或吃到核弹时，Canvas 的摄像机便会以高频衰减的偏移量进行坐标平移 (`ctx.translate`)，HUD 侧边栏和屏幕容器也会闪烁红色微光，让游戏打击感极为爆棚。
4. **复古显像管 (CRT Filter) 扫描线开关**：
   我们通过 CSS Linear Gradients 在屏幕前覆盖了一层动态闪烁的电视光波和暗角，完美重现 90 年代大头贴显示器的复古温暖颗粒感。你可以在 HUD 侧边栏随时一键关闭它。
5. **全功能地图编辑器**：
   你只需在主菜单点击“地图编辑器”，就能用鼠标随意涂抹红砖、钢板、水流、草丛、冰面、刷新点。编辑器自带主脑校验（必须放置 1 个核心、1 个 P1 起点、至少 1 个敌人刷新点），支持随时一键**“测试试玩”**并自动利用 `localStorage` 永久缓存，断电不丢失。
6. **7 种强大的强化能量胶囊 (PowerUps)**：
   - **能量护盾 (S - Shield)**：亮蓝色几何多边形外壁，防御一切攻击。
   - **时空停滞 (T - Time)**：停滞所有敌军坦克的 AI 思考与移动，持续 8 秒。
   - **铁甲鹰巢 (B - Base)**：将基地四周的普通红砖临时升级为合金钢板，持续 15 秒。
   - **火力升级 (U - Upgrade)**：拾取后可将坦克升级至“速射激光”、“双子激光”（齐射）以及“聚能核磁”（贯穿钢板）。
   - **核弹轰炸 (N - Nuke)**：核磁引爆！全屏敌人瞬间在漫天绿色/红色的电磁火花中蒸发。
   - **战机备件 (L - Life)**：奖励一台备用坦克。
   - **电磁穿甲 (P - Pierce)**：让当前激光炮直接贯穿钢板，持续 15 秒。

### ⚡ 临时超载强化（时效 15 秒）

* **电磁穿甲 (P - Pierce)** 强化胶囊：
  * **超载效果**：拾取后立即使战机处于 15 秒的“电磁超载穿甲”状态。
  * **战术价值**：在此状态下，无论你当前处于 Tier 1, 2 还是 3，发射的激光都将**临时获得 Tier 4 穿透合金钢板**的能力！若在 Tier 3 双子激光状态下拾取，将射出能够瞬间贯穿钢板的双子激光，火力爆发力无与伦比。
  * **时效控制**：持续 15 秒，结束后自动恢复当前武器阶梯的正常穿甲状态。

### 💡 核心战斗战术与生存指南

1. **射程与掩体**：在 Tier 1 和 Tier 2 阶段，善于利用 **2x2 微型砖墙系统**。你可以用精准的单发子弹削掉砖墙一角，制造极细的弹道，在敌方子弹无法穿过的死角里远距离阻击敌人。
2. **基地防守与 Shovel 联动**：当拾取 **铁甲鹰巢 (B)** 胶囊后，基地四周的红砖会临时化身合金钢板（持续 15 秒）。如果此时玩家持有 Tier 4 武器或者激活了 **电磁穿甲 (P)** 胶囊，请千万注意不要从内侧开火误打掉属于自己的钢铁护盾！
3. **生死存亡的代价**：每次生命值耗尽损失一台备用战机（Respawn）时，武器系统都将重置回 **Tier 1 初始状态**，失去所有的连发与高能强化。在敌军潮水般的包围中维持高阶武器等级，是通往最后一关（Stage 5 - Goliath's Citadel）的致胜密钥。
